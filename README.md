## ⚡ Deploy to GitHub Pages (Step by Step)

### 1. Create a new repository
- Go to [github.com/new](https://github.com/new)
- Name it exactly: `<your-username>.github.io`
  - Example: if your GitHub username is `mnoman`, name it `mnoman.github.io`
- Set it to **Public**
- Click **Create repository**

### 2. Upload the files
- Click **Add file → Upload files**
- Drag and drop `index.html` and `README.md`
- Commit message: `Initial portfolio launch`
- Click **Commit changes**

### 3. Enable GitHub Pages
- Go to **Settings → Pages**
- Under **Source**, select `Deploy from a branch`
- Branch: `main` / folder: `/ (root)`
- Click **Save**

### 4. Your site is live! 🎉
Wait ~60 seconds, then visit:
```
https://<your-username>.github.io
```
